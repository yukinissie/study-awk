# STUDY AWK
## Usage
### hello_world.awk

```sh
echo | awk -f hello_world.awk
```

### list_command_uid.awk
```sh
chmod 755 list_command_uid.awk
ps -ef | ./list_command_uid.awk
```

### fizz_buzz.awk
```sh
chmod 755 fizz_buzz.awk
seq 1 15 | ./fizz_buzz.awk
```
It have test.
```sh
chmod 755 test_fizz_buzz.sh
./test_fizz_buzz.sh
```
